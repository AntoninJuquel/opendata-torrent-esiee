# Open data torrent

## Install

Prerequisites :

- docker
- chromedriver (optional)

``` shell
sudo apt install docker.io
sudo apt install chromium-browser # version above 95
```

Now launch the install script

``` shell
sudo python install.py
```

## Usage

Once you installed the project you just have to do :

``` shell
python live.py
```
